# 月学月厉害

## 概述

自动化主题月学习计划制定与执行系统，支持学习内容每日推送和多维表格进度跟踪。

## 核心工作流程

### 1. 学习主题识别与解析
当用户提供月度学习主题时，系统自动识别月份与主题的对应关系：

```python
# 示例输入识别
用户输入: "4月学红酒，5月学投资理财，6月学社保"
解析结果:
- 4月: 红酒学习
- 5月: 投资理财学习  
- 6月: 社保学习
```

### 2. 详细学习计划生成
对每个月份主题，系统自动拆解为：

#### 月度层面
- 确定月度学习目标和重点内容
- 规划4周（或按实际周数）的学习模块

#### 周度层面
- 每周一个学习主题模块
- 周一至周五：新知识学习
- 周末：复习/休息（可配置）

#### 每日层面
每个学习日包含：
1. **学习主题**：一句话概括当天学什么
2. **核心知识点**：3-5个结构化知识点
3. **配图**：必要时添加图表辅助理解
4. **资料引用链接**：1-3条权威链接

### 3. 每日内容推送
默认每天08:00推送（时间可配置），推送格式：

```
📅 5月12日 投资学习
🎯 主题：读懂市盈率（PE）
📚 核心知识点：
· PE = 股价 ÷ 每股收益，相当于按当前利润回本需要多少年。
· 静态PE看过去，滚动PE（TTM）看最近四季，更及时。
· 低PE ≠ 便宜：要和行业、历史PE比，还要看成长性。
· 高PE可能代表高增长预期，但也可能是泡沫。
🖼️ [配图：不同行业PE对比柱状图]
🔗 参考：https://xxx.com/pe-guide
```

### 4. 多维表格集成
所有学习记录自动写入多维表格，表格结构：

| 字段 | 类型 | 说明 |
|------|------|------|
| 月份 | 单选 | 所属月份 |
| 主题 | 文本 | 学习主题 |
| 周次 | 数字 | 第几周 |
| 日期 | 日期 | 学习日期 |
| 学习主题 | 文本 | 当日学习主题 |
| 核心知识点 | 多行文本 | 结构化知识点 |
| 配图链接 | URL | 配图链接（可选） |
| 引用链接 | URL | 参考资料链接 |
| 完成状态 | 单选 | 未开始/进行中/已完成 |
| 掌握程度 | 单选 | 未掌握/基本掌握/熟练掌握 |

## 内容质量标准

### 简洁性原则
- 每日核心知识点不超过5个
- 每个知识点描述不超过100字
- 总学习内容控制在300-500字
- 避免专业术语堆砌

### 易吸收性设计
- 使用通俗易懂的语言解释专业概念
- 重要概念配有简单明了的图示说明
- 复杂知识采用分步骤拆解方式

### 信息完整性
- 每个知识点都提供权威参考资料链接
- 引用来源明确标注（网站、书籍、研究报告等）
- 学习路径清晰，有明确的前后知识衔接

## 配置选项

### 学习节奏设置
1. **标准模式**（默认）：
   - 周一至周五：新知识学习
   - 周六：知识复习和巩固
   - 周日：休息或项目实践

2. **强化模式**：
   - 每天学习，包括周末
   - 适合集中提升期

3. **轻松模式**：
   - 仅工作日学习
   - 周末完全休息

### 推送时间配置
用户可自定义推送时间：
- 早晨（07:00-09:00）
- 午休（12:00-14:00）
- 晚间（19:00-21:00）

### 多维表格类型
支持多种多维表格平台：
- 飞书多维表格
- Airtable
- Notion Database
- 其他支持API的表格工具

## 技术实现要点

### 计划生成算法
系统根据以下因素生成学习计划：
1. **主题复杂度**：评估知识深度和广度
2. **时间分配**：合理分配每月学习时长
3. **知识递进**：从基础到高级的渐进式学习
4. **实践结合**：理论知识与实践应用相结合

### 内容生成策略
1. **知识提炼**：从权威来源提取核心知识点
2. **图文匹配**：为每个知识点匹配合适的视觉辅助
3. **引用验证**：确保引用链接的可访问性和权威性
4. **格式优化**：确保内容简洁、易吸收、无废话

### 错误处理机制
1. **信息不足**：当主题信息不足时，主动询问用户补充信息
2. **推送失败**：建立重试机制和备用推送渠道
3. **表格同步失败**：先存本地JSON并告警，确保数据不丢失
4. **链接失效**：定期检查和更新引用链接

## 最佳实践建议

### 学习主题选择
1. **相关性**：选择与职业发展或个人兴趣相关的主题
2. **可实现性**：确保在计划时间内能够掌握核心内容
3. **实用性**：选择能够立即应用的知识技能
4. **系统性**：主题之间最好有逻辑关联或递进关系

### 学习过程管理
1. **每日坚持**：即使时间紧张也要完成最小学习量
2. **主动思考**：不仅要接收信息，还要思考应用场景
3. **及时记录**：在学习过程中记录疑问和心得
4. **定期回顾**：每周回顾学习成果，调整学习策略

### 成果评估
1. **知识测试**：每月末进行知识点掌握测试
2. **实践应用**：将学到的知识应用到实际工作中
3. **成果展示**：创建学习成果展示文档或项目
4. **技能认证**：考虑参加相关技能认证考试
